# 식물성 레시피 플랫폼 MVP 기획서 (부담 완화형)

## 1. 브랜드 메시지 (Main Copy)
> **"완벽한 비건보다, 기분 좋은 한 끼 식사를 지향합니다."**
>
> 채식을 꼭 완벽하게 해내야 한다는 압박감은 내려놓으세요.
> 오늘 하루, 당신의 취향에 맞는 가벼운 식물성 한 끼면 충분합니다.

---

## 2. 웹사이트 구조 (3-Tab Structure)

### ① HOME (메인)
- **Hero:** 브랜드 슬로건 (B안) + 감성적인 요리 영상/이미지
- **Curation:** "처음 시작하기 좋은 10분 레시피", "냉장고 속 재료로 뚝딱"
- **CTA:** "나만의 레시피 공유하기" (등록 폼 연결)

### ② RECIPE (탐색)
- **UI:** 카드형 리스트 (이미지 중심)
- **표시 정보:** 요리명, 난이도(🌱 아이콘), 재료(아이콘 픽토그램)
- **필터:** 시간별(5분/10분/20분), 재료별(두부/버섯/면 등)

### ③ ABOUT (철학)
- **Story:** "왜 한 끼인가?"에 대한 운영자의 따뜻한 편지
- **Guide:** 입문자를 위한 대체 식재료 가이드 (예: 우유 대신 두유/귀리유)

---

## 3. 입력자 최적화 : 'Smart-Step' 등록 시스템

작성자가 요리 과정을 기록하는 것이 아니라, **'블록을 쌓는 게임'**처럼 느끼게 구성합니다.

| 단계 | 항목 | 상세 기능 및 UX |
| :--- | :--- | :--- |
| **Step 1~3** | 기본 정보 | 이미지, 제목, 난이도/시간 선택 (기존과 동일) |
| **Step 4** | 재료 선택 | 재료 퀵 버튼으로 식재료를 선택하면 채식 유형이 자동 판별됨 (아래 상세 참고) |
| **Step 5** | 레시피 빌더 | [자동 넘버링 시스템] 적용 |
| **5-1** | 첫 번째 단계 | '1'이라는 숫자가 자동으로 생성되어 있음. 클릭하여 '이미지 업로드' + '짧은 설명' 입력. |
| **5-2** | 단계 추가 | 하단의 [ + 다음 단계 ] 버튼을 누르면 자동으로 '2'번 블록이 생성됨. |
| **5-3** | 순서 편집 | 작성 중 순서가 바뀌면 드래그하여 위치 이동 시 숫자가 자동으로 재정렬 (2번이 1번으로 가면 숫자가 바뀜). |
| **Step 6** | 완성 및 공유 | 입력이 끝나면 인스타 캡션용 텍스트로도 자동 변환해주는 '복사' 기능 (옵션). |

### #(Update: 채식 유형 자동 판별)

#### 1. 핵심 철학 (업데이트)
- 사용자가 채식 용어를 몰라도 요리할 수 있게 돕는다.
- 재료 선택에 따라 시스템이 채식의 종류를 자동으로 분류하여 정보의 정확성을 높인다.

---

#### 2. 스마트 레시피 입력 시스템 (Smart-Input UI)

입력자가 재료를 선택함에 따라 아래 로직에 의해 **[채식 유형 배지]**가 실시간으로 노출됩니다.

##### [채식 자동 판별 로직]
| 재료 포함 여부 | 최종 표시 유형 | 아이콘 |
| :--- | :--- | :--- |
| 식물성 재료만 섭취 (고기, 생선, 달걀, 우유, 꿀 등 제외) | **비건 (Vegan)** | 🥦 |
| 유제품(우유, 치즈 등) 포함 / 달걀, 고기 제외 | **락토 (Lacto)** | 🥗 |
| 달걀 포함 / 유제품, 고기 제외 | **오보 (Ovo)** | 🍳 |
| 유제품 + 달걀 모두 포함 / 고기 제외 | **락토-오보 (Lacto-Ovo)** | 🧀 |
| 생선/해산물 포함 / 육류 제외 | **페스코 (Pesco)** | 🐟 |
| 육류 또는 가공육 가끔 포함 | **플렉시테리언 (Flexi)** | 🍗 |
| 과일, 견과류, 씨앗류 중심 | **프루테리언 (Fruitarian)** | 🌱 |
| 가공식품 최소화 + 통곡물 중심 | **WFPB (통곡물 채식)** | 🌾 |

##### [입력 UX 상세]
1. **재료 퀵 버튼:** 자주 사용되는 동물성/식물성 식재료를 아이콘 버튼으로 제공하여 클릭 한 번으로 분류 완료.
2. **실시간 가이드:** "우유가 포함되어 '락토' 레시피로 분류되었습니다"라는 안내 문구 노출로 사용자 교육 효과 제공.
3. **인스타 캡션 복사:** 입력이 완료되면 해당 채식 유형 태그와 숫자가 매겨진 레시피 단계를 인스타그램 캡션용 텍스트로 자동 생성.

---

#### 3. RECIPE 상세 화면 UI 구성 (업데이트)

- **이미지 슬라이드:** 상단에 조리 과정 및 완성 사진 슬라이드.
- **자동 넘버링 카드:** `1 클릭 -> 글/이미지 업로드 -> + 버튼 -> 2번 자동 생성` 구조로 입력된 단계별 가이드.
- **상태 배지:** 요리 제목 옆에 시스템이 판별한 채식 유형 아이콘과 난이도(새싹 🌱) 노출.
- **한 끗 노트:** 대체 식재료 및 작성자의 꿀팁 노출.
