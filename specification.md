# 식물성 레시피 플랫폼 MVP 기획서 (부담 완화형)

## 1. 브랜드 메시지 (Main Copy)
> **"완벽한 비건보다, 기분 좋은 한 끼 식사를 지향합니다."**
>
> 채식을 꼭 완벽하게 해내야 한다는 압박감은 내려놓으세요.
> 오늘 하루, 당신의 취향에 맞는 가벼운 식물성 한 끼면 충분합니다.

---

## 2. 웹사이트 구조 (3-Tab Structure)

### ① HOME (메인)
- **Hero:** 브랜드 슬로건 (B안) + 감성적인 요리 영상/이미지
- **Curation:** "처음 시작하기 좋은 10분 레시피", "냉장고 속 재료로 뚝딱"
- **CTA:** "나만의 레시피 공유하기" (등록 폼 연결)

### ② RECIPE (탐색)
- **UI:** 카드형 리스트 (이미지 중심)
- **표시 정보:** 요리명, 난이도(🌱 아이콘), 재료(아이콘 픽토그램)
- **필터:** 시간별(5분/10분/20분), 재료별(두부/버섯/면 등)

### ③ ABOUT (철학)
- **Story:** "왜 한 끼인가?"에 대한 운영자의 따뜻한 편지
- **Guide:** 입문자를 위한 대체 식재료 가이드 (예: 우유 대신 두유/귀리유)

---

## 3. 입력자 최적화 : 'Smart-Step' 등록 시스템

작성자가 요리 과정을 기록하는 것이 아니라, **'블록을 쌓는 게임'**처럼 느끼게 구성합니다.

| 단계 | 항목 | 상세 기능 및 UX |
| :--- | :--- | :--- |
| **Step 1~3** | 기본 정보 | 이미지, 제목, 난이도/시간 선택 (기존과 동일) |
| **Step 4** | 레시피 빌더 | [자동 넘버링 시스템] 적용 |
| **4-1** | 첫 번째 단계 | '1'이라는 숫자가 자동으로 생성되어 있음. 클릭하여 '이미지 업로드' + '짧은 설명' 입력. |
| **4-2** | 단계 추가 | 하단의 [ + 다음 단계 ] 버튼을 누르면 자동으로 '2'번 블록이 생성됨. |
| **4-3** | 순서 편집 | 작성 중 순서가 바뀌면 드래그하여 위치 이동 시 숫자가 자동으로 재정렬 (2번이 1번으로 가면 숫자가 바뀜). |
| **Step 5** | 완성 및 공유 | 입력이 끝나면 인스타 캡션용 텍스트로도 자동 변환해주는 '복사' 기능 (옵션). |

- **대체 재료 툴팁:** "비건 치즈" 클릭 시 → "일반 체다 치즈로 바꿔도 맛있어요!" 안내
- **가벼운 카운터:** "오늘까지 총 0,000개의 식물성 한 끼가 모였습니다."
